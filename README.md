# 🚀 AI Startup Planner

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Ollama](https://img.shields.io/badge/LLM-Ollama-green)
![License](https://img.shields.io/badge/license-MIT-yellow)

AI-powered CLI tool that generates **startup ideas, development roadmaps, and technology stacks** using Large Language Models.

This project helps developers and students quickly turn an idea into a **structured startup execution plan**.

---

# ✨ Features

- 💡 Generate unique startup ideas
- 🗺️ Create a simple MVP development roadmap
- 🧠 Suggest a practical technology stack
- ⚡ Runs locally using **Ollama + LLaMA models**
- 🖥️ Simple and fast **terminal interface**

---

# 🧠 Workflow

```
User Idea
   ↓
AI Startup Ideas
   ↓
Product Roadmap
   ↓
Technology Stack
```

The tool converts a simple idea into a **startup execution blueprint**.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/TheShivaji/ai-startup-planner.git
cd ai-startup-planner
```

Create virtual environment

```bash
python -m venv .venv
```

Activate environment

Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

Make sure **Ollama is running** with the required model.

Then run:

```bash
python main.py
```
